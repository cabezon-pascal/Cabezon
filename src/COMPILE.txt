1. About this archive
The file contained in this archive file was named Cabezon.
The compiler source for the Pascal subset.

2. To try compiling
The execution format of Cabezon Ver0.08 is required to compile the compiler itself.
Turbo Pascal Ver5.5 or later is required to compile the compiler driver.
Compile as follows.

A>TPC CAB     Compiling the compiler driver
A>CAB PHASH   Compiling a complete hash function generator
A>MAKE        Compiling the compiler itself

CAB.EXE is the compiler driver and CAB1.EXE is the compiler itself.
